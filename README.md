# Website-URL-Detection

Dataset: Siddhartha,M. “Malicious URLs dataset,” Kaggle. (https://www.kaggle.com/datasets/sid321axn/malicious-urls-dataset)  

Συνολικά 5 αρχεία:

- my_new_data.csv = δεδομένα με όλα τα εξαγώμενα χαρακτηριστικά (12 στήλες) -- Είναι αρκετά μεγάλο σε μέγεθος για να ανέβει, αλλά μέσα από το κώδικα στο urls_ML.ipynb, δημιουργείται --
- urls_ML.ipynb = κώδικας για την εξαγωγή των νέων χαρακτηριστικών και κώδικας για μοντέλα Machine Learning
- urls_DL_part1.ipynb, urls_DL_part2.ipynb, urls_DL_part3.ipynb = κώδικας για μοντέλα Deep Learning
